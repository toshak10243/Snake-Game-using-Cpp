# 🐍 Snake Game in C++(Turbo Cpp)

A simple **Snake Game** built using **C++ (OOPS, Data Structures, and Multithreading)**, compatible with **Turbo C++**.

## 🚀 Features
- Classic Snake Game with food mechanics 🍏
- Move using **arrow keys**
- Score tracking 🎯
- **No external libraries** required – runs on **Turbo C++**
- Simple and lightweight

## 🖥️ How to Run (Turbo C++)
1. Open **Turbo C++**
2. Create a **new file** and **paste the code** from `SnakeGame.cpp`
3. **Compile** the code (`Alt + F9`)
4. **Run** the game (`Ctrl + F9`)

## 🎮 Controls
- **Arrow Keys** → Move the snake  
- **'X' Key** → Exit the game  

## 🛠️ Technologies Used
- **C++ (Object-Oriented Programming)**
- **Data Structures (Arrays)**
- **Multithreading (Delay Function)**
- **Turbo C++**
